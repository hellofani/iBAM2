పుణ్యుఁడు రామచంద్రుఁ
శ్రీరామచంద్రులవారికి సీతాదేవితో కల్యాణం అయినతరువాత పరిస్థితుల ప్రాబల్యంవలన పట్టాభిషేకం భంగమైపోయింది. దశరథుడు తనమూడవ భార్యకు ఇచ్చిన వరాలనుబట్టి రాముడు అరణ్యాలకు పోవలసి వచ్చింది.
ఉత్పలమాల
పుణ్యుడు రామచంద్రు డట వోయి ముదంబున గాంచె దండకా
రణ్యము, దాపసోత్తమ శరణ్యము, నుద్ధత బర్హిబర్హలా
వణ్యము, గౌతమీ విమల వాః కణ పర్యటన ప్రభూత సా
ద్గుణ్యము, నుల్లస త్తరు నికుంజ వరేణ్యము నగ్రగణ్యమున్.
భావము +/-
పుణ్యమూర్తి శ్రీరామచంద్రుడు అలా అడవులలోనికి ప్రవేశించాడు. అందులో ఒక ప్రత్యేకప్రదేశం దండకారుణ్యం. అది చాలాగొప్ప తపస్సుచేసిన మునీశ్వరులకు నివాసస్థలం. అక్కడ చక్కగా పెంపొందిన నెమళ్ళు పురులువిప్పుకొని నృత్యం చేస్తున్నాయి. గోదావరి నదిలోని స్వచ్ఛమైన నీటితుంపురుల కదలికలతో ఏర్పడిన గొప్పచల్లదనం నిలువెల్లా నింపుకొన్నట్టిది ఆవనం. పెద్దపెద్దచెట్లూ, నేలమీది పొదలూ ఆనందపారవశ్యంతో ఊగులాడిపోతున్నాయి. ఈ అన్నింటితో కలసి ఆ అరణ్యం అరణ్యాలన్నింటికీ అగ్రగణ్యం అయి ప్రకాశిస్తున్నది.
తాత్పర్యం +/-
పుణ్యుడు = పుణ్యాత్ముడు; రామచంద్రుడు = రాముడనెడి చల్లనివాడు; పోయి = వెళ్ళి; ముదంబునన్ = సంతోషముతో; కాంచెన్ = దర్శించెను; దండకారణ్యమున్ = దండకారణ్యమును; తాపస = ఋషులలో; ఉత్తమ = శ్రేష్ఠులకు; శరణ్యమున్ = అండనిచ్చెడిది; ఉద్ధతన్ = జతకట్టిన; బర్హి = ఆడనెమలి; బర్హ = మగనెమలుతో; లావణ్యమును = మనోజ్ఞమైనది; గొతమీ = గోదావరీ; విమల = స్వచ్ఛమైన; వాఃకణ = నీటిబిందుల; పర్యటన = వ్యాప్తులచేత; ప్రభూత = అతిశయించి; సాద్గుణ్యమున్ = సద్గుణసంపత్తిగలది; ఉల్లసత్ = గొప్ప; తరు = చెట్లు; నికుంజ = పొదలచే; వరేణ్యమున్ = ఉత్తమమైనదానిని; అగ్రగణ్యమున్ = గొప్పగ ఎంచదగినదానిని.
