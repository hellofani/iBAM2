తనువు మనువు
భక్తులు చాలా మహిమ కలవారయ్యా ! వారికి లోకసంబంధమైన బంధాలు ఏమీ ఉండవు. ఒక్క నాయందు మాత్రమే సంబంధం ఉంటుంది వారికి. అట్టివారి విషయంలో నేను ఎలా ఉంటానో గమనించు – అంటున్నాడు శ్రీహరి దుర్వాసునితో …
ఆటవెలది
తనువు మనువు విడిచి తనయుల చుట్టాల
నాలి విడిచి సంపదాలి విడిచి
నన్నె కాని యన్య మెన్నడు నెఱుగని
వారి విడువ నెట్టివారి నైన.
భావము +/-
వారికి దేహంతో ముడిఉండదు. కన్నకొడుకులను, కట్టుకొన్న భార్యనీ, చుట్టాలనూ, సంపదలను అన్నింటినీ విడిచివేస్తారు. నన్నుతప్ప మరి దేనినీ వారుకోరరు. అటువంటివారు ఎట్టివారైనాసరే నేను విడిచిపెట్టను.
తాత్పర్యం +/-
తనువున్ = దేహమును; మనువున్ = జీవితాన్ని; విడిచి = వదలి; తనయులన్ = పిల్లలను; చుట్టాలన్ = బంధువులను; ఆలిన్ = భార్యను; విడిచి = వదలివేసి; సంపద = సంపదలు; అలి = అన్నిటిని; విడిచి = వదిలేసి; నన్న = నన్నుమాత్రము; తప్పించి = తప్పించి; అన్యము = ఇతరము; ఎన్నడున్ = ఎప్పుడు; ఎఱుగని = తెలియని; వారిన్ = వారిని; విడువన్ = వదలిపెట్టను; ఎట్టి = ఎలాంటి; వారిన్ = వారు; ఐనన్ = అయినప్పటికిని.
